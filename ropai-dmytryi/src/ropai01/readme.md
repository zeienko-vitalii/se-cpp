# Стекові об'єкти. Конструктор, деструктор, відображення, передача {#mainpage} 
 
## Мета
 
Навчитись створювати об'єкти. Отримати розуміння створення об'єкта 
на стеку а також передачу об'єкти по значенню.
 
## 1. Індивідуальне завдання 
 
Створити клас даних Window та класс відображення даних - WindowViewer. Об'єкт відображення конструюється на стеку функції main() об'єктом даних, що заздалегіть створений на стеку. Передавати Window як значення. Window має всі публічні поля та методи. WindowViewer лише виконує відображення даних у формі <назва поля>=<значення>; всі його методи та атрибути публічні.

# Стекові об'єкти. Конструктор, деструктор, відображення, передача {#mainpage} 
 
## Цель 
 
Навчитись створювати об'єкти. Отримати розуміння створення об'єкта 
на стеку а також передачу об'єкти по значенню.
 
## Задание 
 
Продемострировать использование программного комплекса _Doxygen_ для автодокументирования исходного кода. 
 
## Заключение 
 
Комментарии в стиле __Javadoc__ используются: 
1. В заголовках файлов One.h и main.cpp. 
2. В файле One.h:    
- при объявлении класса One;    
- при объявлении методов:      
- One::One();      
- One::~One();      
- One::setX();      
- One::getX();    
- при объявлении поля One::x. 
3. В файле main.cpp при объявлении функции main(). 
 
Обычные комментарии C++ используются: 
1. В файле One.cpp при реализации методов класса One.
2. В файле main.cpp в коде функции main().

## 1. Індивідуальне завдання 
 
Створити клас даних Monitor та класс відображення даних - Screen. Об'єкт відображення конструюється на стеку функції main() об'єктом даних, що заздалегіть створений на стеку. Передавати Monitor як значення. Monitor має всі публічні поля та методи. Screen лише виконує відображення даних у формі <назва поля>=<значення>; всі його методи та атрибути публічні.

# Тема 5: Абстрктні класт, інтерфейси, серіалізація. {#mainpage}

## Мета

Навчитись застосовувати інтерфейси для роботи класів на прикладі задачі серіалізації.

## 1. Загальне завдання
Реалізувати для кожного із класів даних своєї ієрархії можливість збереження 
та завантаження даних за допомогою класу FileStorage, який видається 
до лабораторної роботи у вигляді бібліотеки.
Показати у звіті бінарний дамп збереженого файлу та відмітити дані із 
власних об'єктів.

# Тема 1: Стекові об'єкти. Коструктор, декструктор, відображення, передача. {#mainpage}

## Мета

Навчитись створювати об'єкти. Отримати розуміння створення об'єкта на стеку, а також передачу об'єкта по значенню.

## 1. Загальне завдання
	Створити клас даних Data1  та клас відображення даних View1. Об’єкт відображення конструюється на 
стеку функції main() об’єктом даних, що заздалегідь створений на стеку. Передавати Data1  як значення.<br>
	Data1 має всі публічні поля та методи. 
	View1 лише виконує відображення даних у форматі <назва поля> = <значення>; всі його методи та атрибути публічні.
	### Прикладна область
	Згідно з варіантом №3, прикладною обастю є «Мишки». 
	Назва Data1  Маніпулятор, поля Data1 : кількість кнопок, тип (мишка, джойстик), у якості відображення даних View1 є екран монітора (Screen).

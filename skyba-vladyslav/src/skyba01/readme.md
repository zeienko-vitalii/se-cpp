# Тема 1: Стекові об'єкти. Конструктор, деструктор, відображення, передача

## Мета
Навчитись створювати об'єкти. Отримати розуміння створення об'єкта на стеку а також передачу об'єкта по значенню.
## 1 ІНДИВІДУАЛЬНЕ ЗАВДАННЯ
Створити клас даних Room та класс відображення даних - Screen. Об'єкт відображення конструюється на стеку функції main() об'єктом даних, що заздалегіть створений на стеку. Передавати Room як значення. Room має всі публічні поля та методи. Screen лише виконує відображення даних у формі <назва поля>=<значення>; всі його методи та атрибути публічні.
